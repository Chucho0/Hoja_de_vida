Lenguajes de programación

Assembly

Assembly es un lenguaje de programación de bajo nivel que permite controlar directamente el hardware de la computadora.

C

C es un lenguaje de programación de alto nivel. Es muy eficiente y se puede usar para crear una amplia variedad de programas.

C++

C++ es una extensión de C que agrega características de programación orientada a objetos.

Haskell

Haskell es un lenguaje de programación funcional. Es muy utilizado en investigación.
